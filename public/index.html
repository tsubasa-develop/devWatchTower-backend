DevWatchTower Backend

開発者向けの更新情報（GitHub Releases / 技術ブログなど）を定期取得し、Supabase に保存し、読み取り専用 API として提供するバックエンドです。

API 仕様: docs/api-doc.yaml (OpenAPI)

何をしている？

  • 外部ソースからコンテンツを取得
  • 統一フォーマットに変換
  • Supabase へ UPSERT
  • 公開 API で取得できるようにする

主なエンドポイント

  • GET /api/contents 公開コンテンツ一覧
  • GET /api/contents/{id} コンテンツ詳細

データ取得（プラグイン）

  • plugins/github GitHub Releases など
  • plugins/line-techblog RSS フィード

ローカル実行

  • npm run dev 開発サーバー
  • npm run sync:releases 取得→DB同期
# contents 一覧（例）
curl "http://localhost:3000/api/contents?limit=20&sort=published_at&order=desc"

# 特定コンテンツ（例）
curl "http://localhost:3000/api/contents/<id>"
このページは簡易なプロジェクト説明用の静的 HTML です。Next.js の挙動により / がこのファイルを直接返さない場合があります。
